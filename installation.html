Using Red Deer in a Maven Project in Eclipse

In you Eclipse based IDE you can install RedDeer from one of these p2 update sites (Help->Install New Software):

For for latest stable version (1.0.1) - for Mars

http://download.jboss.org/jbosstools/updates/stable/mars/core/reddeer/1.0.1/

For latest nightly development version built from master (1.1.0-SNAPSHOT) - for Neon

http://download.jboss.org/jbosstools/neon/snapshots/builds/reddeer_master/

Older versions

For for stable version 0.8.0 - for Luna

http://download.jboss.org/jbosstools/updates/stable/luna/core/reddeer/0.8.0/

For for stable version 0.7.0 - for Luna

http://download.jboss.org/jbosstools/updates/stable/luna/core/reddeer/0.7.0/

For for stable version 0.6.0 - for Luna

http://download.jboss.org/jbosstools/updates/stable/luna/core/reddeer/0.6.0/

For for stable version 0.5.0 - for Luna

http://download.jboss.org/jbosstools/updates/stable/luna/core/reddeer/0.5.0/

For for stable version 0.4.0 - for Kepler

http://download.jboss.org/jbosstools/updates/stable/kepler/core/reddeer/0.4.0/

For for stable version 0.2.0 - for Juno

http://download.jboss.org/jbosstools/updates/stable/juno/core/reddeer/0.2.0/

Then, in your project's MANIFEST.MF file, add Red Deer to the list of required bundles for example:

Require-Bundle: org.jboss.reddeer.swt

Use "reddeer" filter to see what RedDeer plugins you can use in your tests.
